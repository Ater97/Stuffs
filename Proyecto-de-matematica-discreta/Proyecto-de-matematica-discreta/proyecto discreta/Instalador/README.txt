PROYECTO ISOMORFISMO

Instrucciones de instalacion:
* En la carpeta "Instalador" encontrara un archivo .exe, llamado "setup"
* Debera oprimir el icono y aprobar la instalacion.

Instrucciones de uso:
* Una vez concluida la instalacion se abrira el programa en una consola.
* Ingrese los paths de los archivos que desea verificar; como se indica en el manual de usuario.
* Se recomienda leer el manual de usuario antes de iniciar.
